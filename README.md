# Jest Unit Tests

Проект с юнит-тестами для функции `calculateBonus` с использованием Jest.

## Описание

Функция `calculateBonus(a, b)` вычисляет бонус на основе суммы двух чисел:
- Если сумма ≤ 50, возвращает сумму
- Если сумма > 50, возвращает максимум 50

## Установка

1. Клонировать репозиторий:
```bash
git clone https://github.com/olgavasiljeva/architecture-node-tasks.git
cd architecture-node-tasks
```

2. Установить зависимости:
```bash
npm install
```

## Запуск тестов

Для запуска всех тестов:
```bash
npm test
```

```
.
```

## Покрытие тестами

- ✅ Сумма меньше 50
- ✅ Сумма равна 50
- ✅ Сумма больше 50
- ✅ Работа с нулевыми значениями
- ✅ Работа с отрицательными числами
- ✅ Граничные случаи
